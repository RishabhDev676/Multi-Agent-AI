🌐 Multi-Agent AI Workspace

A simple but functional multi-agent orchestration system where four AI agents (Planner, Analyzer, Executor, Referee) work together to produce more accurate and verified outputs than a single chatbot.
The system includes a Flask backend, HTML/JS frontend, and OpenRouter-based model calls, all built using free, open-source models.

🚀 Features :- 

4-Agent Pipeline
Planner → Generates a structured step-by-step plan
Analyzer → Reviews, critiques, and improves the plan
Executor → Produces useful outputs based on the improved plan
Referee → Checks all outputs for contradictions, mistakes, or hallucinations
